# REPOSITORY FOR ALGORITHMS

## Introduction

This repository contains solutions and explanation for common algorithms.

## Setup

Install dependencies

### yarn

```bash
yarn
```

### npm

```bash
npm install
```

### run tests

```bash
yarn jest
```

or

```bash
npx jest
```
